# Supanotes
Supanotes is a notes application that is stored online. This project built using flutter, supabase, and getx.

![Banner](https://github.com/triagung128/supanotes/blob/main/assets/banners/cover.png)

## Feature Apps
- Login
- Register
- Create/Update Notes
- Save Notes Online
- My Profile
- Change Password
- Logout

## State Management
- Flutter Getx

## Dependencies
- get
- google_fonts
- intl
- supabase_flutter
